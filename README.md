# COVID-19 Global Analysis 📊🌍

**Project Goal:**  
Analyze global COVID-19 trends — confirmed cases, deaths, mortality rate trends, and vaccination progress — using SQL, Python, and Tableau.

---

## 📁 **Project Structure**

covid19-global-analysis/

├── raw_data/ # Original OWID COVID data

├── cleaned_data/ # Cleaned dataset

├── sql/ # SQL queries for trends & KPIs

├── notebooks/ # Jupyter Notebook with analysis & plots

├── tableau/ # Tableau dashboard file (.twbx)

├── screenshots/ # Screenshots of final dashboards

└── README.md # This file


---

## ✅ **Key Tools Used**

- **Python:** pandas, numpy, plotly
- **SQL:** PostgreSQL for storing & querying
- **Tableau:** Interactive dashboards with filters, maps, KPIs

---

## 📌 **Key Insights**

- **Total global cases:** 775 million+
- **Total global deaths:** 7 million+
- **Total global vaccinations:** 13.7 billion+
- Mortality rate dropped from ~2.3% (2020) to ~0.9% (2023+)
- Asia leads global vaccination rollout

---

## 📈 **Visuals**

Includes:
- Global daily new cases
- Global mortality rate trend
- Top 10 countries by cases & deaths
- Global vaccination progress
- Interactive world map of total cases
- KPI tiles for quick glance

---

## 🚀 **How to Reproduce**

1. Clone or download this repo
2. Use the SQL scripts to recreate the tables
3. Run the Jupyter Notebook to see step-by-step cleaning & EDA
4. Open the Tableau file for interactive dashboards
